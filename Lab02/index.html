CommandPurpose
git tag 1.0 <commit id>Create a tag marking a specific commit as a release version.
git reset <filename>Unstage a file while keeping changes in the working directory.
git stash listDisplay the list of stashed changes saved temporarily.
git stash popApply the most recent stash and remove it from the stash list.
git rm -f <filename>Force remove a file from both the working directory and Git.
git rebase -i HEAD~3Interactively modify, reorder, or squash the last 3 commits.
